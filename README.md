Welcome to the repository for The Open-Source AI Paradox, a highly interactive, cinematic, and purely web-based presentation deck. This single-page application is designed to deliver a compelling 10-slide narrative exploring the economic and technical dynamics of open-source versus proprietary AI models.  Crafted specifically for presenters Mrinal (Roll #30) and Biman (Roll #19), this deck abandons traditional presentation software in favor of a customized, code-driven storytelling experience.  ✨ Key FeaturesCinematic UI/UX: The interface features a deep-space aesthetic complete with a continuous 3D animated tunnel background, glassmorphism content cards, ambient drifting particles, and a glowing, "breathing" orb element.  Built-in Presenter Engine: Pressing a hotkey toggles a hidden presenter-overlay designed for live speaking. This layout includes a live presentation timer, current and next slide previews, and detailed speaker notes for every single slide.  Zero Dependencies: The entire presentation is built using pure, vanilla HTML, CSS, and JavaScript. It requires no external libraries or build tools, relying only on Google Fonts (Space Grotesk and Inter) for its modern typography.  Immersive Details: Includes a custom neon-orange cursor with an animated trailing ring, pure CSS floating animations (cardFloat), and a fully animated SVG robot graphic.  Responsive & Accessible: Features simulated fullscreen fallbacks for restrictive browsers, a dynamic progress bar, clickable HUD navigation dots, and swipe-gesture support for mobile touch devices.  ⌨️ Controls & NavigationThe deck includes a built-in event listener for seamless keyboard and touch navigation:  Arrow Right / Spacebar: Advance to the next slide.  Arrow Left: Return to the previous slide.  F: Toggle Fullscreen mode.  P: Toggle the Presenter Mode overlay.  Escape: Close Presenter Mode.  Mobile/Tablet: Swipe left or right across the screen to navigate between slides.  📖 Content OverviewThe presentation systematically breaks down the AI ecosystem through 10 distinct, highly-styled slides:  The Hook: Introduces the paradox of spending millions on compute just to give the intelligence away.  The Divide: Compares closed "Walled Gardens" (like GPT-4 and Gemini) against open "Challengers" (like Llama 3 and Mistral).  Trade-Offs: Explores the pros and cons of sharing intellectual property in the AI space.  Trust Gap: Highlights transaction costs and hallucination liabilities in critical fields like healthcare and law.  Monetization Play 1: Details the "Infrastructure Play" and the cost of scaling cloud GPUs.  Monetization Play 2: Discusses enterprise support, SLAs, and zero-tolerance for IP leakage.  Monetization Play 3: Showcases turning projects into products using AI guide personas named Niel and Vidya.  Stakeholders: A data grid revealing who benefits (students, indie devs) and who is disrupted (wrapper apps).  Financing: Analyzes the shift in Venture Capital towards deep-tech infrastructure over generic API wrappers.  Resolution: Concludes that the future is hybrid, where open-source serves as the foundation and proprietary models govern the regulated edge.  🚀 Usage InstructionsBecause this project is entirely vanilla, there is no setup, installation, or build step required.  Simply clone the repository and open the index.html file in any modern web browser to launch the presentation natively.
